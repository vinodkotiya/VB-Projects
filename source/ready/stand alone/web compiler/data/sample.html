Each one of those is called a tag. There is a starting tag and a closing tag. To make a
   
      closing tag just add a /(backslash) to the starting tag. Most, but not all tags have a closing tag.
   
      
Think of tags as talking to the browser, or better yet giving it instructions. What you
   
      have just told the browser is 'this is the start of a HTML document' <HTML> and 'this is the end of a HTML document' </HTML>. 
       So remember every opening tag <TAG>must have (not4all) a closing tag </TAG>       
   
      
       Now we see some other tags which are used in webpages . Every tag works differently
hence according to your nessecity use these tags       INSIDE Body       The function of these tags are also specified in right hand side.

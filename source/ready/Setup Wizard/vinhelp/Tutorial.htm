Tutorial

Let us create a setup.

We will create a setup for windows notepad application so first go to c:\windows directory search the notepad.exe file , copy it and paste it in a directory <volume(e)>:\stand alones\webcompiler\ . now rename it to My App.exe

Note that <volume(e)>:\stand alones\webcompiler\ will be known as Source Dir\ for your application My App.exe

You can also choose any other exe instead of notepad.

Now open the New Project in VIN Setup Wizard and go to Step1

Step1:

  1.Application Name : Enter My App as application name.

  2. Version : Enter 1.0 for version

  3. Company Name: Enter VINSOFT as company Name

 

 4. Welcome splash screen to be displayed on installer at startup : It may be a text message or any image file. we will choose image as a welcome message so click on Image option .

To Open an image file click on browse '...' button a window will open select all files and open the file wel.gif from wizardpath\data\images\wel.gif

 here wizardpath is the path of the VIN Setup Wizard's Default Folder where it is placed or installed.

  5. Left Side Display / Logo on your Setup: Similar to 4 click on image option and open an image file left.jpg from wizardpath\data\images\left.jpg

Step1 Completed . Click on Next>>

 

Step2:

 1. Software Information : Make the check box true. Click on Browse '...' Button and open the file text.txt from wizardpath\data\text.txt however you can type your own in the text box .

 2. Lisence Agreement : Make the check box true. Click on Browse '...' Button and open the file text.txt from wizardpath\data\text.txt however you can type your own in the text box .

 3. Read Me : Make the check box true. Click on Browse '...' Button and open the file text.txt from wizardpath\data\text.txt however you can type your own in the text box .

Step2 Completed . Click on Next>>

 

Step 3:

1. Source Directory : It is the directory where your application's exe and other data files and subfolders are placed. eg our application My App.exe is placed inside

  e:\stand alones\webcompiler\

then select the source directory like this

2. System Files : Your application may need some system files generally dll or ocx files which helps your application to run on other systems.These files are generally installed to system or system32 folder of windows. however you can specify any other folder by typing in the dropdown box.

We don't need any system files for My App.exe So untouch this area.

3. Installation Directory : It is the directory where your application and its data file and its subfolders will be installed.Type C :\Program Files\VINSOFT

4. Create Shortcuts: we will create shortcuts in Desktop and Programs group for My App.exe

  Desktop : To create shortcuts on Desktop give name of shortcut relative to Desktop eg \My App . Also specify Link with File relative to your source directory eg My App.exe then click on Create Button shown in figure below.

it will create shortcut my app on desktop which open the My App.exe

  Programs : Select Programs From Drop Down . In Name of Shortcut type \vinsoft\My App

  and in Link With File again type My App.exe

it will create shortcut startmenu-> Programs -> vinsoft -> My App which open the My App.exe

Step3 Completed . Click on Next>>

 

Step4 :

1.Show System Administrator Name : Check it to true.

2.Show System Administrators Company Name : Check it to true.

3. Registration Code: Check it to true. and click the Generate button . A random registration code will be shown write down this registration code in a paper because it will be needed on installation time.

Step4 Completed . Click on Next>>

 

Step5 :

1. Prompt For Readme.txt : Uncheck the check box.

2. Prompt For Launch Application : make true the checkbox. click on unchecked button from Checked / UnChecked .

Now specify the path of application file relative to Source Directory .

Since our Source Dir is e:\stand alones\webcompiler so only type My App.exe as shown below in fig.

3. Run this Program in background : Unchek it .

4. Prompt For Reboot the Computer : Uncheck it.

 5. Finishing splash screen to be displayed on installer at End of setup : It may be a text message or any image file. This time we will choose a text message so click on message option and type Thanking You .......

Now select any foreground color and make background tranceparent.

Now Save your project by clicking on Save Button.

Now click on compile button :- If there is any error then first remove them .

Now Click on Build Button :- This will build your setup in the output directory specified by you.

Click on Run Button :- To Run the setup

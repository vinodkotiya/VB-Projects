Step4 ->> System Information

This information will shown when registering the application during installation.

1.Show System Administrator Name : Shows the computers owner name on which application is installing.

2.Show System Administrators Company Name : Shows the computers owner company name on which application is installing.

3. Registration Code: Enter the registration Code for your application which will be asked to user at installation time and without right registration code setup will not proceed.

On clicking the Generate button you can make random registration codes.

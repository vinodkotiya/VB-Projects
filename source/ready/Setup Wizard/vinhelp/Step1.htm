Step1 ->> Welcome Screen

 This screen takes the primary information about setup which would be shown when executing the setup.

1.Application Name : Enter your application name.

2. Version : Enter applications version

3. Company Name: Your company Name

4. Welcome splash screen to be displayed on installer at startup : It may be a text message or any image file. you can choose any one.

  •   Message : Enter the welcome message

  . Background Color : determines the background color of message.

  .Tranceparent : No background color of message.

  .Foreground Color : determines the foreground color of message.

  • Image : Specify the path of image file. Resolution of the image file should be nearly 360 X 330 .

  • Time : Determines the no of seconds that welcome message or image will be shown.

5. Left side Display / Logo On Your Setup:- It may be also a text message or any image file.

  •   Message : Enter the Left side message

  . Background Color : determines the background color of message.

  .Tranceparent : No background color of message.

  .Foreground Color : determines the foreground color of message.

  • Image : Specify the path of image file. Resolution of the image file should be nearly 360 X 330 .

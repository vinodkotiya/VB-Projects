Step5 ->> Finishing Screen

These informations will be shown when setup is finishing.

1. Prompt For Readme.txt : If you want that your readme file (which should be included in step 2) will be prompt to open then make true. If you want it should be checked or not checked then click on appropriate button Checked / UnChecked

2. Prompt For Launch Application : If you want that your main application file will be prompt to run then make true. If you want it should be checked or not checked then click on appropriate button Checked / UnChecked .

Also specify the path of application file relative to Source Directory (as specified in step3)

3. Run this Program in background : Sometimes you want some another process to run in background for special purpose such that registering application or scanning the plugins. Then specify the path of application file relative to Source Directory (as specified in step3)

4. Prompt For Reboot the Computer : Some application need the system to restart to complete the installation. Check it if you want reboot.

5. Finishing Screen : Similar to welcome screen in step1.

6. Output Directory :- It is the Directory Where your setup will be generated.

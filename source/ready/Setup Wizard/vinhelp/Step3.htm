Step3 ->> Application Information

Enter you applications information.

1. Source Directory : It is the directory where your application's exe and other data files and subfolders are placed. eg if application webcompiler.exe and its data files and folders are placed inside

  e:\stand alones\webcompiler\

then select the source directory like this

2. System Files : Your application may need some system files generally dll or ocx files which helps your application to run on other systems.These files are generally installed to system or system32 folder of windows. however you can specify any other folder by typing in the dropdown box.

  so first open the system file then select or type path where to install. Now click on add the file will be added and embedded to your setup . These files will be installed to the desired path when setup run on any machines.

3. Installation Directory : It is the directory where your application and its data file and its subfolders will be installed.

4. Create Shortcuts: You can create shortcuts in Desktop , Start up and Programs group

  Desktop : To create shortcuts on Desktop give name of shortcut relative to Desktop . Also specify Link with File relative to your source directory then click on Create Button.

 eg Desktop \my app and Source Dir\ My App.exe

will create shortcut my app on desktop which open the My App.exe

  Programs : Select Programs From Drop Down then Similar to Desktop

  Startup : Select Startup From Drop Down then Similar to Desktop

 

  Edit : To edit any shortcut first select it Then Click on Edit button. Modify the shortcut and then click on Update Button.

  Delete: To remove any shortcut click on Delete Button.
